# El Gran Coliseo
Plataforma open-source para torneos de Dota 2 construida con Next.js 14 App Router y Tailwind CSS v4.

Demo: https://el-gran-coliseo.netlify.app
Repo: https://github.com/Alecwce/el-gran-coliseo

## Stack
- Next.js 14, React 19, TypeScript
- Tailwind CSS
- Deploy en Netlify

## Desarrollo local
pnpm install
pnpm dev
